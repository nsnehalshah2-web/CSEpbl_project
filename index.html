Problem Statement

Early detection of cardiovascular diseases remains a global challenge due to complex clinical markers. Our project addresses the lack of accessible, real-time diagnostic tools for medical practitioners in high-pressure environments.

Research Gap

Existing solutions often require high-end infrastructure or manual interpretation. CardioAI leverages optimized Random Forest algorithms to provide 92%+ accuracy on limited clinical datasets with instant web execution.

System Methodology

Dataset

UCI Heart Disease Dataset (900+ instances). Cleaned via mean-imputation and One-Hot encoding.

Model

Random Forest Classifier with 100 estimators. Scaled using StandardScaler for numerical stability.

API Stack

FastAPI Backend with Pydantic validation and asynchronous request handling.
